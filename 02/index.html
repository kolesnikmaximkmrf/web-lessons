Форматування тексту

Тег <strong> визначає важливість визначеного тексту. Зазвичай виділяє текст напівжирним.

Тег <em> визначає важливість визначеного тексту. Зазвичай виділяє текст курсивом.

Тег <mark> визначає важливість визначеного тексту. Зазвичай виділяє текст жовтим фоном.

Тег <small> робить текст меншим.

Тег <ins> визначає важливість визначеного тексту. Зазвичай виділяє текст, підкреслюючи його.

Тег <del> визначає важливість неважливість визначеного тексту. Зазвичай виділяє текст, закреслюючи його.

Теги <sub> та <sup> відображають текст у вигляді нижнього та верхнього індексів, наприклад, x1 = x2.

Розмітка коду

Тег <code> використовується для відображення коду, при відображенні шрифт тексту стає monospace-ом.

Тег <pre>, зазвичай, використовується з тегом <code> та використовується для відображення пробілів та іншої розмітки тексту.

Приклад використання тегів <pre> та <code>:


            void arrOut(char arr[], int n) {
                for (int i = 0; i < n; i++) {
                    cout << setw(3) << i;
                }
            
                cout << endl;
            
                for (int i = 0; i < n; i++) {
                    cout << setw(3) << *(arr + i);
                }
            
                cout << endl << endl;
            }

Тег <kbd> використовується для відображення кнопки, яку користувач повинен натиснути, або тексту, який користувач повинен внести.

Тег <samp> використовується для відображення виводу.

Тег <var> використовується для відображення змінної x.

Цитати та визначення

<dfn> - тег, який використовується для визначень.

Тег <abbr> використовується для відображення абревіатури, наприклад, КМРФ.

Тег <q> використовується для відображення цитат: Все ж таки вона крутиться!

Тег <cite> використовується для посилання на джерело: Галілей Все ж таки вона крутиться!

Тег <blockquote> використовується для виділення цитати:

За одну ночь нельзя изменить жизнь. Но за одну ночь можно изменить мысли, которые навсегда изменят твою жизнь.

Розриви

Тег <br> використовується для розриву строк:

Так ніхто не кохав. Через тисячі літ
лиш приходить подібне кохання.
В день такий розцвітає весна на землі
І земля убирається зрання…

Тег <hr> використовується для лінії між текстом та чимось ще.

Списки

Тег <ol> використовується для створення нумерованого списка.

Тег <ul> використовується для створення маркованого списка.

Тег <li> використовується для створення елементів списка.

Тег <dl> використовується для створення списка визначень.

Тег <dt> використовується для створення визначень у списку.
